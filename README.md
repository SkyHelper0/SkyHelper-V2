## Commands

- /skyhelper - Opens the main GUI. (Alias is /st)
- /skyhelper config - Opens the GUI to edit the config.
- /skyhelper help - Displays the various commands and their usages
- /skyhelper reload <data/mayor/slayer> - Forces skyhelper to refresh of data.
- /skyhelper fetchur - Displays the current Fetchur item for the day.
- /skyhelper griffin refresh - Forces a refresh for the Griffin burrow waypoints.
- /skyhelper aliases - Opens the GUI to edit command aliases.
- /skyhelper editlocations (/skyhelper editlocation, gui, loc) - Opens the GUI to modify HUD element locations.
- /skyhelper shortcuts - Opens the GUI to modify keybind shortcuts.
- /skyhelper spam - Opens the GUI to modify spam filters.
- /skyhelper enchant - Opens the GUI to customize enchant names.
- /skyhelpercata <player> - Shows information about a player's Catacombs statistics.
- /skyhelperslayer <player> - Shows information about a player's Slayer statistics.
- /sthw <set/remove/clear/help> <x y z> <name> - Allows to set waypoints while in the Crystal Hollows.
